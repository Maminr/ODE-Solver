# Differential Equation Solver (حل‌گر معادلات دیفرانسیل)

## English Version

### Project Description

This is a Python-based GUI application for solving Ordinary Differential Equations (ODEs). Users can define differential equations of any order and view their analytical solution, solution plot, and directional field (for first-order equations).

### Requirements

1. Python 3.8 or newer must be installed.
2. Required libraries can be installed via:

```bash
pip install customtkinter sympy pillow numpy matplotlib
```

> It's recommended to use a virtual environment such as `venv` or `conda`.

### How to Run

To run the program, simply execute:

```bash
python ode-solver.py
```

(Make sure the filename matches your main Python file.)

### Features

- Solve ODEs of any order (homogeneous and non-homogeneous)
- View both textual and graphical analytical solutions
- Plot the solution with custom constant values
- Visualize directional field for first-order ODEs
- Dual language interface (English / Persian)
- Help section with usage guide

### Required Files

Ensure the following files are in the same directory:

- `ode-solver.py` (main application script)
- `logo.png` (optional program logo)

## نسخه فارسی

### معرفی پروژه

این پروژه یک رابط گرافیکی (GUI) برای حل معادلات دیفرانسیل معمولی (ODE) با استفاده از زبان **پایتون** است. کاربران می‌توانند معادلات با مرتبه دلخواه وارد کرده و پاسخ تحلیلی، نمودار راه‌حل و میدان جهتی (برای معادلات مرتبه اول) را مشاهده کنند.

### پیش‌نیازها

۱. نصب Python (نسخه 3.8 یا بالاتر)  
۲. نصب کتابخانه‌های موردنیاز با دستور:

```bash
pip install customtkinter sympy pillow numpy matplotlib
```

> توصیه می‌شود پروژه در محیط مجازی (مثل `venv` یا `conda`) اجرا شود.

### اجرای برنامه

پس از نصب پیش‌نیازها، برای اجرای پروژه کافی‌ست:

```bash
python ode-solver.py
```

(در صورتی که نام فایل اصلی چیز دیگری است، همان را بنویسید.)

### قابلیت‌ها

- حل معادلات دیفرانسیل همگن و ناهمگن با هر مرتبه
- نمایش پاسخ به صورت متنی و تصویری
- امکان رسم نمودار با مقادیر دلخواه ثوابت
- نمایش میدان جهتی برای معادلات مرتبه اول
- رابط کاربری دو زبانه (فارسی / انگلیسی)
- بخش راهنما برای توضیح نحوه استفاده

### فایل‌های موردنیاز

فایل‌های زیر باید در کنار فایل اصلی باشند:

- `ode-solver.py`
- `logo.png` (در صورت استفاده)
